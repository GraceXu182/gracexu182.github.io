I am currently a Postdoctoral Associate at the [Center for Brains, Minds and Machines](https://cbmm.mit.edu/), [McGovern Insitute for Brain Research](https://mcgovern.mit.edu/) at MIT, I am supervised by [Prof. Tomaso Poggio](https://poggio-lab.mit.edu/people/tomaso-poggio) and [Prof. George Karnidakis]() in the Division of Applied Mathematics, Brown University. My interests include fundamental new developments on neural networks (e.g., optimization, graph embeddings) as well as applications of machine learning. My objective is to develop effective data-driven machine learning algorithms and frameworks (neural network-based Gaussian embedding, deep Graph Convolutional Networks, U-Net, deep CNN, uncertainty quantification) for diverse applications with emphasis on biomedicine. For example, my recent focus at MIT has been on developing graph embedding methods to analyze the dynamic functional brain network analysis based on fMRI or MEG data as well as multi-omics data for Alzheimer’s disease. Other projects include diverse medical image computing and analysis problems, and clinical problems, e.g., automated heterogeneous RBC classification for sickle cell disease.
